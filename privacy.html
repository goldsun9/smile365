1 기본 원칙
✅ 뇌팔청춘365는 사용자의 개인정보를 수집, 저장, 공유, 판매하지 않습니다.

본 앱은 회원가입이 없으며, 서버로 전송되는 데이터가 없습니다. 앱 내 모든 활동 기록(점수, 미션, 구매 여부 등)은 오직 사용자의 기기 내부 저장소에만 보관됩니다.

3 마이크 권한 사용
🎙️ 마이크 접근 목적
  • 발성 훈련 시 음성 볼륨(데시벨) 측정에만 사용됩니다
  • 음성 내용은 절대 녹음, 저장, 전송되지 않습니다
  • 오직 소리의 크기(loudness)만 실시간으로 측정 후 즉시 폐기됩니다
  • 마이크 권한을 거부해도 다른 기능은 정상 이용 가능합니다

마이크 권한은 웹 표준 기술인 Web Audio API를 사용하며, 수집된 음성 데이터는 외부 서버로 전송되지 않습니다.

1 Overview
✅ Laugh 365 does NOT collect, store, share, or sell any personal information.

This app requires no account registration and transmits no data to any server. All in-app activity data (scores, missions, purchase status, etc.) is stored exclusively on the user's local device.

3 Microphone Permission
🎙️ Why We Need Microphone Access
  • Used only to measure voice volume (decibel level) during vocal exercises
  • Voice content is never recorded, stored, or transmitted
  • Only real-time loudness is measured and immediately discarded
  • All other features remain fully functional without microphone permission

Microphone access uses the standard Web Audio API. No audio data is sent to any external server.
